# Icon 图标

## 基本用法 

```html
<cap-icon icon="bluetooth" size="medium" />

<cap-icon icon="bluetooth" size="40px"></cap-icon>
```

## API

### Props 


|  参数   | 说明  | 类型 | 默认值 | 版本
|  ----  | ----  | ---- | ---- | -
| icon | icon名称 | string | `''` | -
| size | icon大小 | string / `large` `medium` `small`  | `medium` | -
